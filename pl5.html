Что такое CSS​

CSS — это инструмент, который помогает сделать сайт красивым и удобным. Если представить сайт как дом, то HTML — это его стены и фундамент (структура), а CSS — это обои, краска, мебель и декор (внешний вид).  С помощью CSS можно:  ​

    выбрать цвета текста и фона;  настроить шрифты и их размеры; расположить элементы (например, картинки или кнопки) в нужных местах  Сделать сайт удобным для просмотра на телефонах и планшетах; 

Проще говоря, CSS отвечает за то, как сайт выглядит, чтобы он был приятным и понятным для пользователей. Однако даже с CSS сайт может выглядеть скучно. Пользователям нравится интерактивность: возможность что-то "пощупать", взаимодействовать с элементами, наблюдать динамику. Когда речь заходит о таком функционале, в дело вступает JavaScript.​
